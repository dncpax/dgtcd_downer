about= <div style="max-width:33em">A plugin for the <a href="https://cdd.dgterritorio.gov.pt/">DGT CDD Portal</a>

	<ul><li><strong>PT</strong> - Este plugin adiciona uma ferramenta ao QGIS Processing Toolbox, para descarregar vários conjuntos de dados geoespaciais (LiDAR, MDS, MDT) do portal português DGT CDD. Permite a autenticação, divide grandes áreas em partes mais pequenas, organiza os ficheiros descarregados e pode, opcionalmente, criar e carregar VRTs para dados raster.

        Para executar eficazmente o plugin é necessário registar-se em <a href="https://cdd.dgterritorio.gov.pt/">https://cdd.dgterritorio.gov.pt/</a>.

        Ferramenta desenvolvida pela comunidade e não afiliada com a Direção-Geral do Território!</li>

	<li><strong>EN</strong> - This plugin adds a tool to the QGIS Processing Toolbox to download various geospatial datasets (LiDAR, MDS, MDT) from the Portuguese DGT CDD Portal. It handles authentication, divides large areas into smaller chunks, organizes downloaded files, and can optionally create and load VRTs for raster data.

        To effectively run the plugin you'll need to register at <a href="https://cdd.dgterritorio.gov.pt/">https://cdd.dgterritorio.gov.pt/</a>.

        Tool developed by the community and not affiliated with Direção-Geral do Território!</li></ul>


    </div>
